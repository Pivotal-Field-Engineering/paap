# paap - Platform as a Product CLI

Command line interface demo harness for Pivotal's Platform Engineer workshop.

## Getting Started

### Clone the github repo and run
```
go install
paap help
``` 

### Prerequisites

* Golang
* Cloud Foundry CLI
* Set environment variables denoted in "paap help"
